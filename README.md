# CourseSphere

Sistema full stack para gerenciamento de cursos e lessons.

## Tecnologias

### Backend
- Node.js
- Express
- SQLite
- JWT

### Frontend
- React
- Vite
- Axios

## Funcionalidades

- Registro de usuários
- Login JWT
- CRUD de cursos
- CRUD de lessons
- Dashboard
- Rotas protegidas

## Como rodar

### Backend

```bash
npm install
npm run dev

### Frontend

npm install
npm run dev
